Front end

Front end : partie visible d’un site web, celle que voit et utilise l’utilisateur (design, boutons, textes).
  • HTML : HTML est un langage qui sert à structurer le contenu d'une page web (titres, textes, images, liens, etc.).

  • CSS : CSS est un langage qui permet de styliser une page web (couleurs, polices, marges, dispositions, etc.).

  • Javascript : Javascript est un langage qui permet de rendre les pages web interactives (animations, clics, formulaires, etc.).

Back end

Back end : partie cachée qui gère les données, les comptes, les serveurs et les bases de données.
  • Python : Python est un langage de programmation simple et puissant, utilisé pour créer des sites web, des applications, de l’automatisation, de l’analyse de données, et plus encore. Dans mon cas, je l'ai plus travaillé pour des algorithme (type calculer une moyenne, trier par ordre croissant, etc...)

  • PHP : PHP est un langage de programmation utilisé principalement pour créer des sites web dynamiques côté serveur.

  • MySql : MySQL est un système de gestion de base de données qui permet de stocker, organiser et récupérer des informations pour des sites ou des applications.
